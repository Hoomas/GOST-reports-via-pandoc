## Подраздел первого уровня

Inline-формула: $E = mc^2$. Выключная нумерованная:

\begin{equation}\label{eq:einstein}
E = mc^2,
\end{equation}

где $E$ — энергия, $m$ — масса, $c$ — скорость света. Ссылка: формула \eqref{eq:einstein}.

# Блоки кода

Пример Python-кода:

```python
def fibonacci(n):
    if n < 2:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

for i in range(10):
    print(fibonacci(i))
```
