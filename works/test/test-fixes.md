# Введение {.unnumbered}

Это тестовый документ для проверки исправлений в шаблоне. Он содержит длинные слова вроде «термоэлектродвижущей», «высокотемпературного» и «электроэнцефалография», чтобы проверить перенос. Также здесь есть цитата [@test2024].

# Раздел с формулой

Уравнение Эйнштейна имеет вид:

\begin{equation}
E = mc^2,
\end{equation}

где $E$ — энергия, $m$ — масса, $c$ — скорость света.
